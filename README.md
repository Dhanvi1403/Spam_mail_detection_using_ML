# AI Spam Mail Detector

An intelligent Machine Learning based system that detects whether an email is Spam or Ham (Not Spam) using Natural Language Processing techniques and classification algorithms.

---

## Overview

Spam emails are one of the most common cybersecurity and communication problems. This project uses Machine Learning techniques to automatically classify emails as spam or legitimate messages.

The system is trained using text-processing techniques and ML models from Scikit-learn to improve email filtering efficiency.

---

## Features

- Spam vs Ham email classification
- Machine Learning based prediction
- Text preprocessing using NLP
- Fast and accurate detection
- Beginner-friendly implementation
- Easy to customize and improve

---

## Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Natural Language Processing (NLP)
